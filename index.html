Автоматическая сегментация узелков в легких

Разработка нейросети на базе архитектуры U-Net для точного выделения легочных узелков на КТ-снимках.

О проекте

Задача: Точное выделение узелков

Обнаружение и точная сегментация (выделение границ) узелков в легких на КТ-снимках — ключевой этап в диагностике и мониторинге заболеваний, включая рак легких. Ручная сегментация требует времени и может варьироваться у разных специалистов.

Решение: U-Net для сегментации

Создание и обучение нейронной сети на основе архитектуры U-Net, специально разработанной для задач сегментации биомедицинских изображений. Цель — разработать инструмент, который сможет:

  • Автоматически и точно определять границы узелков.
  • Значительно ускорить процесс анализа КТ-снимков.
  • Предоставлять объективные, воспроизводимые результаты сегментации.
  • Стать полезным инструментом для радиологов и исследователей.
Как это работает?

Технологический процесс

Основные этапы разработки и обучения модели U-Net для сегментации.

1. Данные

Использование анонимизированных КТ-снимков с экспертной разметкой (масками) узелковых образований.

2. Предобработка

Нормализация интенсивности пикселей, аугментация данных для повышения устойчивости модели.

3. Архитектура U-Net

Применение архитектуры U-Net, состоящей из сжимающего (энкодер) и расширяющего (декодер) путей.

4. Обучение

Тренировка сети с использованием функций потерь, оптимизированных для сегментации (например, Dice Loss).

5. Оценка

Валидация модели на отложенной выборке с использованием метрик качества сегментации (Dice score, IoU).

6. Оптимизация

Подбор гиперпараметров и возможное улучшение архитектуры для достижения наилучших результатов.
